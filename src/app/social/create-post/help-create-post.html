Creación de posts

Permisos requeridos

Administradores:

Un administrador no requiere permisos del Módulo Social para postear en ninguna página, lo único que necesita es ser administrador de la página (o que la pagina le pertenezca), que la pagina esta agregada en el Módulo Social en la ventana de Administración y que este activa.

Empresarios:

Un empresario requiere ser administrador de la página (o que la pagina le pertenezca) y que un administrador le otorgue permisos sobre esa página en concreto, adicionalmente dicha página debe estar activa.

Funciones según el tipo de usuario

Administradores:

Un administrador puede realizar tanto "Posteos Admin" como "Posteos Pendientes".

Empresarios:

Un empresario solo puede realizar "Posteos Pendientes".

¿Cómo funciona la creación de posts?

La siguiente lista enumera la secuencia correcta para realizar cualquier tipo de post:

  • Escribe el contenido o texto que deseas que tenga el post en el recuadro blanco, no puedes postear un post sin texto.

  • Selecciona una fecha para para obtener el feedback de Facebook, no puedes postear sin seleccionar una fecha valida. Esta es una sugerencia para indicar cuando deseas que se obtenga la retroalimentación de Facebook.

  • Selecciona una imagen para mostrar en el post, su tamaño no debe ser mayor a 10Mb.

  • Procede a realizar el tipo de post que desees y para el que poseas permisos.

  • Independientemente de tu elección al hacer clic en ambas opciones de posteos se abrirá una ventana de Facebook en la que deberás seleccionar sobre que páginas deseas otorgar permisos a Intelitur, tus páginas se desplegaran, selecciona las que desees, y te pedirá permisos. Esto solo pasara la primera vez puesto que Facebook recordará la configuración en futuras acciones, sin embargo, siempre puedes cambiarla fácilmente cuando se desee (se recomienda agregar todas tus páginas).

  • Una vez otorgues los permisos sobre la o las páginas, podrás ver como se muestra una ventana de selección donde puede elegir una o más páginas para postear, el post será idéntico en todas las páginas no importa cuantas páginas se elija.

  • La información que ingresaste no se limpiara de los campos, puedes hacer múltiples versiones de un mismo post para que los administradores seleccionen la más adecuada en caso de realizar un Posteo Pendiente.
