Gestión de publicaciones

Permisos requeridos

Administradores:

Un administrador no requiere permisos del Módulo Social para hacer uso de las funciones, pero si necesita ser administrador de la página sobre la que quiere administrar en Facebook, esto es necesario para "Publicar", "Ver Post" y "FeedBack". No es necesario para "Eliminar".

Empresarios:

Un empresario requiere ser administrador de la página (o que la pagina le pertenezca) y tener permisos en el Módulo social sobre esa página. Solo puede "Ver Post".

Funciones según el tipo de usuario

Administradores:

Un administrador puede publicar un post pendiente, eliminar un post pendiente, visualizar posts y obtener el feedback de Facebook, para las páginas en las que sea administrador en Facebook.

Empresarios:

Un empresario solo puede visualizar posts publicados por sigo mismo, para las páginas en las que sea administrador en Facebook.

¿Cómo funciona eliminar un post pendiente?

La siguiente lista enumera la secuencia correcta para eliminar un post pendiente:

  • Solo puedes eliminar post pendientes, para eliminar un post pendiente solo debes hacer clic en "Eliminar". Un post pendiente eliminado no puede ser posteado, sin embargo, puedes ver el post en "Ver Post".

¿Cómo funciona ver un post?

La siguiente lista enumera la secuencia correcta para ver posts:

  • Cualquier tipo de post pendiente o no puede ser visualizado (si eres administrador). Debes hacer clic en "Ver Post".

  • Se abrirá una ventana de Facebook, la página a la que pertenece el post debe estar seleccionada en los permisos otorgados a Intelitur, no importa la cantidad de páginas que tengas asociadas a Intelitur.

  • Terminado el paso anterior podrás visualizar el post.

¿Cómo funciona publicar un post pendiente?

La siguiente lista enumera la secuencia correcta para publicar un post pendiente:

  • Debes hacer clic en "Publicar".

  • Se abrirá una ventana de Facebook, la página a la que pertenece el post debe estar seleccionada en los permisos otorgados a Intelitur, no importa la cantidad de páginas que tengas asociadas a Intelitur.

  • Terminado el paso anterior podrás visualizar una ventana en la que debes elegir únicamente la página en la que el post fue publicado y clic en continuar.

  • No se valida si la página esta activa o no en la aplicación, puesto que cuando se realizó el post pendiente la página si estaba activa.

¿Cómo funciona obtener el feedback de Facebook?

La siguiente lista enumera la secuencia correcta obtener el feedback de Facebook:

  • Debes hacer clic en "FeedBack".

  • Se abrirá una ventana de Facebook, la página a la que pertenece el post debe estar seleccionada en los permisos otorgados a Intelitur, no importa la cantidad de páginas que tengas asociadas a Intelitur.

  • Terminado el paso anterior podrás visualizar una ventana en la que debes elegir únicamente la página en la que el post fue publicado y clic en continuar.

  • Podrás visualizar las reacciones actualizadas en "Ver Post".
