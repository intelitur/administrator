Gestión de permisos y páginas

Permisos requeridos

Administradores:

Un administrador no requiere permisos del Módulo Social para realizar ningún tipo de acción en esta ventana.

Empresarios:

Un empresario NO puede acceder a esta ventana.

Funciones según el tipo de usuario

Administradores:

Un administrador puede agregar permisos a usuarios empresarios sobre distintas páginas, también puede deshabilitar el permiso. Un administrador puede agregar páginas a la aplicación y permitir así su uso a empresarios o administradores, así mismo puede deshabilitar una página temporalmente.

Empresarios:

Ninguna.

¿Cómo funciona el agregar una página?

La siguiente lista enumera la secuencia correcta para agregar una página:

  • Haciendo clic en el botón con un signo de "+" en la sección de páginas (+).

  • Se abrirá una ventana de Facebook en la que deberás seleccionar sobre que páginas deseas otorgar permisos a Intelitur, tus páginas se desplegaran y selecciona las que desees y te pedirá permisos. Esto solo pasara la primera vez puesto que Facebook recordara la configuración en futuras acciones, sin embargo, siempre puedes cambiarla fácilmente cuando se desee.

  • Una vez otorgues los permisos sobre la o las páginas, podrás ver como se muestra una ventana de selección donde puedes elegir una página para agregar al Modulo Social, solo puedes seleccionar una y esta no debe estar ya agregada en la plataforma.

¿Cómo funciona el agregar un permiso a un empresario?

La siguiente lista enumera la secuencia correcta para dar permisos a un usuario:

  • Haciendo clic en el botón con un signo de "+" en la sección de permisos (+).

  • Se abrirá una ventana en la cual navegaras entre todos los usuarios de la aplicación, debes seleccionar uno.

  • A continuación, veras todas las páginas disponibles en el Módulo Social, debes seleccionar una para indicar que otorgas permisos a ese usuario para esa página.

  • Seguidamente podrás ver una ventana de confirmación para verificar tu selección, de ser correcta has clic en confirmar.
